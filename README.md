# Projeto-de-especializa-o-em-Cyberseguran-a
Este projeto foi desenvolvido como parte do **Bootcamp de Cybersegurança da DIO**, em parceria com a **Riachuelo**. O objetivo é demonstrar a aplicação prática de conceitos de segurança digital, utilizando o **NotebookLM** como um segundo cérebro especializado em Cybersegurança.
